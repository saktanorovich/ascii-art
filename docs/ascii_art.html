ASCII Art

ASCII art is the art of creating pictures with a grid of ASCII characters. We are interested in the most primitive style, where overall character density represents differently shaded areas of the picture.

You should write a proof‑of‑concept program that renders a filled closed polygon with a rectangular grid of ASCII characters. The process is explained below.

Character Mapping

Each pixel is represented by an ASCII character depending on the percentage of its filled area:

Pixel percentage filled Character Glyph ASCII code
0% to <25%Full stop.46
25% to <50%Plus sign+43
50% to <75%Small letter oo111
75% to <100%Dollar sign$36
100%Number sign#35

Example Illustration

Input

The input contains several test cases. Each begins with integers n, w, and h (number of vertices, width, and height). Then follow n lines with polygon vertex coordinates in clockwise order.

Output

For each test case, output h lines with w ASCII characters representing the drawing. Separate consecutive cases with a blank line.

Sample Input

6 8 7
7 6
1 0
1 7
5 5
2 4
2 3
